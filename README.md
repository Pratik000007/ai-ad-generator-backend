📬 API Documentation (Core Endpoints)
Auth Flow
POST /api/auth/register - Create a new account.

POST /api/auth/login - Authenticate and receive a JWT.

POST /api/auth/forgot-password - Trigger reset link to Gmail.

POST /api/auth/reset-password - Update password using a valid token.

Ad Management
POST /api/ads/generate - Generate AI ad content (Requires JWT).

GET /api/ads/my-ads - Retrieve ads created by the logged-in user.

Admin Tools
GET /api/admin/dashboard - System-wide analytics and user management (Admin Only).
